# 🏥 Hospital Emergency Room Dashboard

## 📌 Project Overview

This project presents an interactive **Hospital Emergency Room Dashboard** developed using **Power BI**. The dashboard applies **descriptive analytics** to historical Emergency Room data to provide meaningful insights into patient volume, admissions, waiting times, patient satisfaction, demographics, and department referrals.

The objective of this project is to support operational monitoring and data-driven decision-making through interactive visualizations.

---

## 🎯 Objectives

- Analyze Emergency Room patient data.
- Monitor hospital operational performance.
- Visualize patient demographics and referral patterns.
- Identify patient arrival trends.
- Support data-driven healthcare decisions.

---

## 🛠 Tools Used

- Power BI Desktop

- DAX
- Microsoft Excel

## 📊 Dashboard Features

## 💡 Key Insights

- A total of **9,216 patients** visited the Emergency Room.
- Approximately **4,612 patients** were admitted.
- The average patient waiting time was **35.26 minutes**.
- The average satisfaction score was **4.99**.
- Most patients arrived without a department referral.

---

## 🚀 Recommendations

- Increase staffing during peak hours.
- Improve patient flow to reduce waiting times.
- Monitor patient satisfaction regularly.
- Use historical trends for workforce planning.
- Continue using business intelligence dashboards to support operational decisions.
